EDGAR MARKOV — CARD-DRAW UPGRADE PACKAGE (vampire-for-vampire)
================================================================
Base deck : D1+Imp+Budg-FINAL-100
Problem   : you run ~3 card-draw sources; the premium lists run 8-12.
Method    : swap your weakest NON-drawing vampires for vampires that DRAW.
            Tribe count, Edgar eminence triggers, and board presence stay identical.
Bracket   : NO CHANGE. Adds 0 Game Changers, 0 combos. Stays Bracket 3.
Cost      : core+strong ~ $4.45 total (all cheap singles; not in DSK/INR boxes).

=== CORE (do first) — card draw 3 -> 5 =========================
CUT  Rakish Heir            ->  ADD  Champion of Dusk      (~$0.30)
        why: weakest vampire (1/3 premium runs it) -> ETB draw X = # Vampires you control
CUT  Vicious Conquistador   ->  ADD  Dusk Legion Zealot    (~$0.30)
        why: budget 1-drop (0/3 premium runs it) -> 1-drop, ETB draw a card

=== STRONG (next) — card draw 5 -> 7 ===========================
CUT  Captivating Vampire    ->  ADD  Twilight Prophet       (~$2.50)
        why: clunky lord/steal -> upkeep: reveal top, draw it + drain the table (Ascend)
CUT  Bloodletter of Aclazotz->  ADD  Preacher of the Schism (~$1.35)
        why: narrow drain-doubler -> attack: make a Vampire token OR draw + lose life

=== FLEX (optional, push to 8-9) ===============================
ADD  Idol of Oblivion       (~$1.00)  artifact: T, draw if you made a token this turn
ADD  Dusk Legion Duelist    (~$3.10)  draws when +1/+1 counters land on it (loves Cathars/anthems)
        (cut a remaining marginal card, e.g. Indulgent Aristocrat — you keep 5 sac outlets)

All 4 core/strong CUTS and ADDS are Vampires -> tribe + eminence unchanged.
Resulting dedicated card-draw sources: ~7 (matches the premium Edgar lists).
None of these are pullable from Duskmourn / Innistrad Remastered — buy as singles.
